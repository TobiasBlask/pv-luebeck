# Probevorlesung TH Lübeck 2026

Konzeptioneller Vergleich qualitativer und quantitativer Marktforschungsmethoden in der Dienstleistungsindustrie.

Probevorlesung und Kolloquium für die W2-Professur Allgemeine BWL · Marketing und empirische Marktforschung an der TH Lübeck, FB Maschinenbau und Wirtschaft.

## Inhalt

- 33 Slides Probevorlesung für Bachelor BWL, 2. Semester, 60 Minuten
- 16 Slides Kolloquium
- 16 Backup-Slides für Vertiefungs- und Transferfragen

Bahnreiseerlebnis als durchgehender Fall, drei Akte (quantitativ, qualitativ, mixed methods), interaktive Popover-Erklärungen für zentrale Methodenbegriffe.

## Anzeigen

Statischer HTML-Deck. Lokal anschauen:

```sh
python3 -m http.server 8000
```

Dann `http://localhost:8000/TH_Luebeck_Gesamtpraesentation_Sans_v2.html` öffnen. Pfeil-Tasten zum Navigieren.

## Stack

- Vanilla HTML, CSS, JS
- `<deck-stage>` Web Component für Slide-Skalierung
- Inline-SVG für alle Methodendiagramme
- Inter, Instrument Serif, DM Mono via Google Fonts
